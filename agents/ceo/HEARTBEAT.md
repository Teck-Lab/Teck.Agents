# CEO HEARTBEAT — Run on every activation

## 1. Review Progress
- Check Paperclip board: any blocked tasks older than 24h?
- Review CTO's latest technical plan — on track?
- Check model spend dashboard: within budget across squads?

## 2. Unblock
- Any TL waiting on product decisions?
- Any cross-team conflict (Dev Team vs Infra Team)?
- Any CMO/CFO requests needing CEO sign-off?

## 3. Prioritize
- Review incoming feature requests — what matters most?
- Re-evaluate scope: expand, hold, or reduce?
- Update CTO on priority changes

## 4. Stakeholder
- Review CMO's market analysis — any competitive moves?
- Review CFO's spend report — any budget concerns?
- Any external stakeholder needs (partners, customers)?

## 5. Extraction
- Document decisions made this session
- Update product roadmap if changed
- Extract reusable insights for future planning
