# CMO HEARTBEAT — Hourly timer + wake-on-demand

## 0. Competitive Analysis (Proactive)
- Research top 3 competitors: Shopify, commercetools, BigCommerce — what features shipped recently?
- Compare feature gaps: what do they have that we don't? Create feature proposals for CTO
- Monitor pricing changes: any shifts that affect our positioning?
- Scan HackerNews, ProductHunt, Reddit r/ecommerce for emerging platforms and trends
- Use Librarian + Explorer via OMO to gather data, Oracle to analyze strategic impact

## 1. Market Pulse
- Competitor moves: any new features, pricing changes, acquisitions?
- Market trends: any shifts in e-commerce/platform space?
- Customer feedback: recurring themes from support channels?

## 2. Feature Pipeline (Proactive)
- Compare Teck's feature set against competitor benchmarks → identify gaps
- For each gap: is it a must-have (blocking deals) or nice-to-have (differentiator)?
- Create feature proposal tasks assigned to CTO with priority and competitive justification
- Wake CTO when new proposals are created: POST /api/agents/{cto-id}/wakeup

## 3. Brand Health
- Messaging consistency across channels: any drift?
- Competitive positioning: still differentiated?
- Any PR or community engagement opportunities?

## 4. Growth Metrics
- Customer acquisition: trending up or down?
- Churn signals: any patterns emerging?
- Partnership pipeline: any new opportunities?

## 5. Go-to-Market
- Upcoming feature launches: GTM plan ready?
- Target segments: any new segments to explore?
- Content/positioning needed for next release?

## 6. Extraction
- Document market intelligence findings
- Update competitive landscape analysis
- Wake CEO if strategic shift needed: POST /api/agents/{ceo-id}/wakeup
