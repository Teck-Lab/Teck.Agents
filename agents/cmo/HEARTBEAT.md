# CMO HEARTBEAT — Run on scheduled cadence

## 1. Market Pulse
- Competitor moves: any new features or pricing changes?
- Market trends: any shifts in e-commerce/platform space?
- Customer feedback trends: any recurring themes?

## 2. Brand Health
- Messaging consistency across channels: any drift?
- Competitive positioning: still differentiated?
- Any PR or community engagement opportunities?

## 3. Growth Metrics
- Customer acquisition: trending up or down?
- Churn signals: any patterns emerging?
- Partnership pipeline: any new opportunities?

## 4. Go-to-Market
- Upcoming feature launches: GTM plan ready?
- Target segments: any new segments to explore?
- Content/positioning needed for next release?

## 5. Extraction
- Document market intelligence findings
- Update competitive landscape analysis
- Brief CEO on market position
