# Researcher HEARTBEAT

## 1. Research Queue
- Any pending research tasks from CTO or TLs?
- Prioritize by impact and urgency

## 2. Investigation
- Execute research: search docs, evaluate libraries, prototype
- Every finding: cite source, state confidence level

## 3. Deliverable
- Produce research report with actionable recommendation
- Include "further investigation needed" items if incomplete

## 4. Extraction
- Save findings to team knowledge base
- Update comparison matrices if relevant
- Flag critical findings to CTO immediately
